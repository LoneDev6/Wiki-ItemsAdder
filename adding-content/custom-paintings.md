# Custom Paintings (ItemsAdderAdditions)

Custom paintings are available through [ItemsAdderAdditions](https://itemsadderadditions.com/en/docs/custom-paintings).

{% hint style="warning" %}
Requires **Paper 1.21.5 or higher**.
{% endhint %}

Custom paintings let you register real Minecraft painting variants directly from your ItemsAdder YML files. Players can encounter them in-game and, if configured, vanilla painting items can randomly pick them.

Check the full documentation [here](https://itemsadderadditions.com/en/docs/custom-paintings).
