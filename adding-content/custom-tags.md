# Custom Tags (ItemsAdderAdditions)

Custom tags are available through [ItemsAdderAdditions](https://itemsadderadditions.com/en/docs/custom-tags).

Group content IDs once and reuse the group in supported ItemsAdderAdditions features. Tags are loaded from a top-level `tags` section in any ItemsAdder content YAML file.

Check the full documentation [here](https://itemsadderadditions.com/en/docs/custom-tags).
