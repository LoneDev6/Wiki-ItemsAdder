# Versions Support

{% hint style="danger" %}
Download the version that matches your server's Minecraft version!
{% endhint %}

## ItemsAdder versions

| Minecraft       | ItemsAdder |
|-----------------|------------|
| 26.1.1-26.1.2   | Beta       |
| 1.21.11         | 4.0.16+    |
| 1.21.10         | 4.0.15+    |
| 1.21.6-1.21.8   | 4.0.12+    |
| 1.21.5          | 4.0.10+    |
| 1.21.4          | 4.0.6+     |
| 1.21.3          | 4.0.4+     |
| 1.21.1          | 4.0.2+     |
| 1.20.6          | 4.0.0+     |
| 1.15.2 → 1.20.4 | v3         |

{% hint style="info" %}
MC **26.1.1** and **26.1.2** are supported in the latest beta, available on [Discord](https://discord.gg/4dfnpUK).
{% endhint %}

## Server software

| Software       | Status                  |
|----------------|-------------------------|
| Paper          | ✅ Supported             |
| Spigot         | ✅ Supported             |
| Purpur         | ✅ Supported             |
| Leaf           | ✅ Supported             |
| UniverseSpigot | ✅ Supported             |
| Folia          | ❌ Not supported for now |
| Arclight       | ⚠️ Unofficial           |
| Mohist / Youer | ⚠️ Unofficial           |
| Forge / Fabric | ❌ Not supported         |

{% hint style="info" %}
Any fork of **Spigot** or **Paper** should work with ItemsAdder.
{% endhint %}

{% hint style="warning" %}
**Arclight** and **Mohist / Youer** are hybrid servers. They may work but are not officially supported and bug reports for these platforms won't be accepted.
{% endhint %}
