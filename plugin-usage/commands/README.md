#  驻拽转

* iatexture:
  * 转专: Force the client to reload resourcepack (a player or everyone)
  * 驻 砖砖: /iatexture \[all or player]
  * 砖: "ia.user.iatexture"
* iaget:
  * 转专: Get custom item by namespaced id or by id
  * 驻 砖砖: /iaget \<item> \[amount]
  * 砖: "ia.admin.iaget"
* iagive:
  * 转专: Give custom item to player
  * 驻 砖砖: /iagive \<item> \[amount] \[silent]
  * 砖: "ia.admin.iagive"
* iadrop:
  * 转专: Drop custom item at coords or at player location
  * 驻 砖砖: /iadrop \<item> \[amount]
  * 砖: "ia.admin.iadrop"
* iaremove:
  * 转专: Remove custom item from player inventory
  * 驻 砖砖: /iaremove \<item> \[amount] \[silent]
  * 砖: "ia.admin.iaremove"
* iatag:
  * 转专: Shows custom item debug info
  * 驻 砖砖: /iatag
  * 砖: "ia.admin.iatag"
* iacustommodeldata:
  * 转专: Shows custom item CustomModelData
  * 驻 砖砖: /iacustommodeldata
  * 砖: "ia.admin.custommodeldata"
* iaplayerstat:
  * 转专: Set a custom player stat (and update HUD) value
  * 驻 砖砖: /iaplayerstat \<read|write|increment|decrement> \<player> \<attribute> \<int|float|string> \[value]
  * 砖: "ia.admin.iaplayerstat"
* ia:
  * 转专: Opens items info GUI
  * 驻 砖砖: /ia
  * 砖: "ia.user.ia"
* iarepair:
  * 转专: Repairs current item (vanilla or custom)
  * 驻 砖砖: /iarepair
  * 砖: "ia.admin.iarepair"
* iarename
  * 转专: Rename current item (vanilla or custom). Supports emojis (font\_images) and ItemsAdder text-effects.
  * 驻 砖砖: /iarename \<name>
  * 砖: "ia.admin.iarename"
* iadurability:
  * 转专: Modify durability of current item (vanilla or custom)
  * 驻 砖砖: /iadurability
  * 砖: "ia.admin.iadurability"
* iareload:
  * 转专: Reloads items\_packs files
  * 驻 砖砖: /iareload
  * 砖: "ia.admin.iareload"
* iazip:
  * 转专: Executes /iareload and generates pack.zip file
  * 驻 砖砖: /iazip
  * 砖: "ia.admin.iazip"
* iablock:
  * 转专: Show info about block you're looking at
  * 驻 砖砖: /iablock
  * 砖: "ia.admin.iablock"
* ialiquid:
  * 转专: Show info about liquid you're looking at
  * 驻 砖砖: /ialiquid \[x] \[y] \[z]
  * 砖: "ia.admin.ialiquid"
* e:
  * 转专: Show list of font images (emojis, huds...). Can also be used to send messages with emojis autocomplete.
  * 驻 砖砖: /iaimage
  * 砖: "ia.user.image.gui"
* iarecipe:
  * 转专: Show item recipe GUI
  * 驻 砖砖: /iarecipe \[item]
  * 砖: "ia.user.iarecipe"
* iahud:
  * 转专: Force show/hide a HUD manually by namespaced id
  * 驻 砖砖: /iahud \[name]
  * 砖: "ia.admin.iahud"
* iaconvert:
  * 转专: Converts old 1.0 world to 2.0 world (blocks), you can decide to delete ores (set stone) or keep them
  * 驻 砖砖: /iaconvert \[world] \[deleteores]
  * 砖: "ia.admin.iaconvert"
* iainfo:
  * 转专: Shows info about the plugin
  * 驻 砖砖: /iainfo
  * 砖: "ia.admin.iainfo"
* iakill:
  * 转专: Kills custom mobs
  * 驻 砖砖: /iakill
  * 砖: "ia.admin.iakill"
* iasummon:
  * 转专: Summons custom mobs
  * 驻 砖砖: /iasummon \[amount]
  * 砖: "ia.admin.iasummon"
* iaexport:
  * 转专: Exports a namespace into a .zip file
  * 驻 砖砖: /iaexport \<namespace>
  * 砖: "ia.admin.iaexport"
* iaspawntree:
  * 转专: Spawns a custom tree
  * 驻 砖砖: /iaspawntree
  * 砖: "ia.admin.iaspawntree"
* iaplaytotemanimation:
  * 转专: Shows a Totem Of Undying animation
  * 驻 砖砖: /iaplaytotemanimation
  * 砖: "ia.admin.iatotemanimation"
* iaplaysound:
  * 转专: Play itemsadder sounds. Useful in console since vanilla /playsound is bugged in console.
  * 驻 砖砖: /iaplaysound
  * 砖: "ia.admin.iaplaysound"
* iacleancache:
  * 转专: Cleans unused IDs from cache to allow them to be used by future added blocks / items.
  * 驻 砖砖: /iacleancache
  * 砖: "ia.admin.iacleancache"
* iasha1:
  * 转专: Calculates sha1 of the current resourcepack.
  * 驻 砖砖: /iasha1
  * 砖: "ia.user.iasha1"
* iahitbox:
  * 转专: Shows the hitbox of placed furnitures.
  * 驻 砖砖: /iahitbox
  * 砖: "ia.admin.iahitbox"
* iaconfig:
  * 转专: Main command to execute some operations on the plugin configurations.
  * 砖: "ia.admin.config"
