
### How to retexture the GUI background

```yaml
Title: '%img_offset_-16%%img_blank_menu%'
```

{% hint style="warning" %}
PlaceholderAPI is required!
{% endhint %}

Basically `%img_offset_-16%` is used to shift back the texture `%img_blank_menu%`by **16** pixels.
