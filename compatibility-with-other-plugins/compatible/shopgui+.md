## 如何在此插件内使用 ItemsAdder 物品

{% embed url="https://docs.brcdev.net/#/shopgui/shops-items-setup?id=itemsadder" %}

```yaml
      1:
        type: item
        item:
          itemsAdder: "ruby_sword"
        buyPrice: 50
        sellPrice: 25
        slot: 0
```
